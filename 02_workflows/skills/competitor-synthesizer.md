# Skill: Competitor Synthesizer

## Trigger
When you need to research competitor features, pricing, UX patterns, content strategy, or messaging to inform GrahAI product decisions.

## Outcome
Competitive intelligence brief (4–6 pages) comparing 3–5 competitors to GrahAI on 8 dimensions, with strategic recommendations.

## Context Files
Read first:
- `01_context/product/7_SUCCESS_METRICS.md` - GrahAI's current metrics and targets
- `01_context/product/3_USER_FLOWS.md` - GrahAI user personas
- `01_context/product/4_COPY_SYSTEM.md` - GrahAI positioning vs. competitors

## Primary Competitors (Research these)
- **Astrotalk** (Indian, chat/phone astrology, largest user base)
- **Co-Star** (US, Gen-Z friendly, personalized daily notifications)
- **The Pattern** (US, introspection-focused, minimal design)
- **AstroSage** (Indian, Kundli software, traditional astrology)
- **Kundli Software** (Indian, desktop + web, birth chart computation)

## Steps
1. **Visit each competitor**: Astrotalk.com, Co-Star.com, ThePattern.app, AstroSage.com, KundliSoft.com
2. **Audit 8 dimensions** for each:
   - **Product offering**: Core features (chart, readings, daily updates, consultations, community)
   - **Pricing model**: Freemium/paid, subscription vs. one-time, price points (₹/USD)
   - **User experience**: Onboarding flow, mobile-first vs. web, design system tone
   - **Content strategy**: How they explain astrology, tone (mystical vs. scientific), copy style
   - **Community/engagement**: Social features, user-generated content, retention mechanics
   - **Technical**: Tech stack (if discernible), integrations, API openness
   - **Competitive moats**: What's hard to copy? (Data, community, brand, algorithm)
   - **Gaps vs. GrahAI**: What GrahAI does better, what we're missing
3. **Screenshot key flows**: Onboarding, paywall, chart view, readings
4. **Document pricing**: Create comparison table (product, price, frequency, what's included)
5. **Analyze messaging**: Note competitor copy tone (mystical/scientific/social), headlines, CTAs
6. **Identify trends**: What patterns emerge across 5 competitors? (e.g., all freemium? all mobile-first?)
7. **Spot GrahAI gaps**: Are we missing a feature all 5 competitors have?
8. **Output brief**: Feature matrix, pricing table, UX pattern analysis, messaging benchmarks, 3 strategic recommendations

## Quality Gates
- Must include screenshots/evidence for each claim (no speculation)
- Pricing table must show price, currency, billing frequency, feature differences
- Each competitor must be rated on 1–5 scale across 8 dimensions vs. GrahAI
- Strategic recommendations must be actionable (e.g., "copy Co-Star's onboarding flow" vs. vague "improve UX")

## Edge Cases
- **Astrotalk dominance**: They have 100M+ users; note where we compete vs. where we differentiate
- **Regional differences**: AstroSage targets Hindi users; note language/cultural differences
- **Desktop vs. mobile**: Kundli Software is desktop-first; compare our mobile-native approach
- **Chat astrology**: Astrotalk's real astrologers are different business model than our automated charts; document clearly
